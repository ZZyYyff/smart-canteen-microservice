### 4.1 开发中发现并修复的问题

| 编号 | 问题描述 | 解决方案 |
|------|---------|---------|
| BUG-01 | ErrorCode 缺少 ORDER_STATUS_ERROR 枚举值 | 新增 ORDER_STATUS_ERROR(1002) |
| BUG-02 | maven-compiler-plugin 未配置 -parameters 参数，导致 @PathVariable 反射失败 | pom.xml 添加 `<parameters>true</parameters>` |
| BUG-03 | jackson 序列化 LocalDateTime 缺少 JavaTimeModule | ObjectMapper 添加 `.findAndRegisterModules()` |
| BUG-04 | Result.success("string") 与 Result\<Void\> 返回类型冲突 | Void 返回使用 Result.success() 无参版本 |

## 5. 遗留风险

| 风险 | 等级 | 说明 |
|------|:----:|------|
| 分布式事务 | 中 | 跨服务调用（下单→扣库存）若库存扣减成功但后续创建订单失败，库存已回滚但回滚也可能失败 |
| 数据库单点 | 低 | 单 MySQL 实例，无主从/读写分离 |
| Nacos 单点 | 低 | 单机模式，生产环境建议集群 |
| 测试覆盖 | 低 | 缺少跨服务的集成测试（需要 Nacos/MySQL/Redis 运行环境） |
| 性能测试 | 中 | 未进行压力测试，限流参数需根据实际负载调优 |
